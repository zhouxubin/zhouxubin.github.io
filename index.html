#

登录接口

页面校验规则
  1. 键盘弹出来后，底部的按钮贴在键盘上边
  2. 两个输入框都输入之后才能点击按钮，否则置灰

入参， /tools/userhelper.ashx

参数名 参数类型 是否必传 参数值
action string YES doLogin
username string YES 用户名
password string YES 密码两次md5
token string YES 生成规则：(当前时间戳+NEWREN)MD5后+/当前版本号(例如:1.1.8)这个token在每次运行app就生成一次，之后用同一个token
sys string YES ios/andriod

出参，返回值

字段名 字段类型 备注
status int 接口返回状态0-成功非，小于0表示后台有问题，大于0直接提示message
data id 下边的表格
message string 消息提示，如果是大于0，把message提示出来
字段名 字段类型 备注
address string 地址
amount string 金钱
area string 地区
avatar string 头像
birthday date 生日
email string 电子邮箱
group_id int 会员id
id int 登录人id
nick_name string 昵称
password string 密码，这个密码需要在本地做两次md5之后传给后台
point int 积分
qq string QQ号
sex string 性别
telphone string 电话号码
user_name string 用户名
mobile string 手机号
fans int 粉丝数量
follows int 我的关注
background string 封面图url
Intro string 个人简介
